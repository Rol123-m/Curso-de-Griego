Aprendiendo Griego Koiné: Resumen de Reglas Clave

Resumen de Reglas Clave

1. Tercera Declinación

  • Raíces terminadas en consonante
  • Para hallar la raíz: Usa el genitivo singular y elimina la terminación (ej. σαρκός → σαρκ-)

2. Género

  • Memoriza el artículo con el sustantivo (ej. ἡ σάρξ = femenino)

3. Cuadro de Terminaciones

  • Domina las terminaciones de caso:
    • Nominativo: ς/–
    • Genitivo: ος
    • Dativo: ι
    • etc.

4. Regla 7: Cuadrado de Oclusivas

| Labiales (π, β, φ) + σ → ψ |
| Velares (κ, γ, χ) + σ → ξ |
| Dentales (τ, δ, θ) + σ → σ |

5. Regla 8: La tau (τ) final desaparece

  • Ejemplo: ὀνοματ → ὄνομα

6. Adjetivo πᾶς ("todo")

  • Modelo esencial para participios en el futuro
  • Ejemplo: πᾶς ὁ ἄνθρωπος = "Todo hombre" (posición predicativa)

7. Traducción del Artículo

  • ὁ δέ = "Pero él"
  • Artículo + preposición = atributivo (traducir como relativo)
